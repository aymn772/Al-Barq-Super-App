# البرق للتوصيل (Albarq Delivery)

تطبيق توصيل متكامل فائق الأداء بتصميم عربي عصري.

## المميزات
- نظام طلب ذكي للمطاعم، الصيدليات، والتموينات.
- لوحة تحكم إدارية شاملة.
- نظام دفع عبر المحافظ الإلكترونية.
- تتبع حي للطلبات.
- دعم كامل للغة العربية والإنجليزية.
- وضع ليلي (Dark Mode).

## طريقة التشغيل

1. قم بتثبيت المكتبات:
```bash
npm install
```

2. قم بتشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. لبناء المشروع للإنتاج:
```bash
npm run build
```

## التقنيات المستخدمة
- React
- Vite
- Tailwind CSS
- Firebase (Auth, Firestore)
- Framer Motion
- i18next
